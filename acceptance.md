# SPARK Hand Project Page - Acceptance Criteria

## 必须满足 (Must Have)
1. **所有 10 个板块完整呈现**：Hero, Teaser, Key Contributions, Abstract, Prototype, Method, Experiments, Video, Poster/Slides, BibTeX+Footer
2. **所有图片正确显示**：11 张论文图 + 1 张 Graphical Abstract 图，路径正确，尺寸适配
3. **视频可播放**：本地 MP4 通过 HTML5 `<video>` 标签正常播放，含 controls
4. **外部链接全部可用**：IEEE / arXiv / PDF / Poster / PPT 链接均正确指向
5. **响应式布局**：桌面端（>1024px）和移动端（<768px）均正常显示，无水平溢出
6. **BibTeX 格式正确**：可复制的引用代码块
7. **页面加载速度**：首屏加载 < 3s（本地测试），图片使用合理尺寸

## 应该满足 (Should Have)
8. **Method 折叠交互**：Fig4/Fig6/Fig7 的 Technical Details 区域可展开/收起
9. **实验图片点击放大**：Fig9/Fig10/Fig11 点击后弹出大图
10. **导航栏锚点跳转**：点击导航可跳转到对应板块
11. **移动端汉堡菜单**：navbar 在小屏幕下折叠为汉堡菜单

## 可以有 (Nice to Have)
12. **实验结果 carousel**：除三卡片外，下方增加可选的 carousel 展示
13. **平滑滚动**：锚点跳转带 smooth scroll 动画
14. **图片懒加载**：viewport 外的图片延迟加载
